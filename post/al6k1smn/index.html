写作介绍

写作的要点分为思路和语言两大部分

思路，则包括了正确审题和使用正确的文章结构，从而考察同学们的篇章组织能力，同时作为四六级写作的评分标准，是十分重要的。

总而言之，考点就是：用正确的结构写文章

写作类型

第一步，我们要先保证我们能够正确审题。

那么第二步，就是保证文章结构的正确性和完整性了。

针对这一点，其实就是考察同学们是否能够针对不同类型的文章，使用正确的文章结构。

四六级中常考的有六种作文类型（题型）：

每一种类型都对应了不同的文章结构

  • 现象解释型：问题/现象描述 + 观点论述 + 引出结论
  • 问题解决型：问题/现象描述 + 原因分析 + 问题解决
  • 对比选择型：问题/现象描述 + 观点论述 + 引出结论
  • 图画图标型：图画/图表反映的问题/现象描述 + 观点论述 + 问题解决/展望未来
  • 谚语解释型：谚语解释 + 原因分析/观点论述 + 问题解决/展望未来
  • 书信：写信目的/原因 + 过程描述 + 表达期望与感激之情

文章的结构大体上都是三段式：总—分—总。

  • 第一段：引出/解释题目的观点；现象；谚语；目的
  • 第二段：对观点的看法/选择原因；分析现象产生的原因；为什么谚语这么说的原因；达到目的的理由
  • 第三段：总结观点；引出结论；展望未来；表达期望与感谢

下面针对具体写作类型，具体分析。

对比选择型

  1. 第一段：引出两种现象，做出选择，表达自己的观点
  2. 第二段：针对自己的选择，进行原因分析
  3. 第三段：做出总结，再次重申观点

问题解决型

  1. 第一段：提出问题
  2. 第二段：分析问题产生的原因
  3. 第三段：提出解决问题的措施

书信

  1. 第一段：说明写信目的/原因
  2. 第二段：具体描述过程
  3. 第三段：总结，表示期望与感激之情
